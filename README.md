# Golden Guns 2.1.0
Use Golden Guns unlocked in the Compound in The Dreaming!

## About
This plugin is an improved version of [arti's GoldGunsInDreaming](https://thunderstore.io/c/receiver-2/p/arti/GoldGunsInDreaming/). Like the original plugin, it allows you to use golden guns in The Dreaming as you unlock them in The Compound.

## Features
New features added by me include:
- Classic Mode is now supported
- ConfigurationManager support. Plugin can now be toggled independently for both Classic and RankedProgression.
- Semi-autos now spawn with matching Golden Magazines.
- The Golden Flashlight is now supported.
